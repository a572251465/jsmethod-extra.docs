getExtNameReverse ​

版本 以及描述 ​

V2.0.11

根据 分隔符 拿到文件名称

导入 ​

shell
import { getExtNameReverse } from "jsmethod-extra"

参数 ​

  • name 要分割的字符串
  • split 分割符号, 默认是 .

数据类型 ​

ts
export declare const getExtNameReverse: (
    name: string,
    split?: string
) => string;

使用案例 ​

ts
import {getExtNameReverse} from "jsmethod-extra"

getExtNameReverse("1111"); // "" 
getExtNameReverse("file.text"); // file
getExtNameReverse("fi,le.text", ","); // fi
getExtNameReverse(null); // null
