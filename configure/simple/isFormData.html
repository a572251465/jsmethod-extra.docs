isFormData ​

V2.0.0

判断 value 值的数据类型是否为 formData 类型

导入 ​

shell
import { isFormData } from "jsmethod-extra"

数据类型 ​

ts
export declare function isFormData(value: unknown): boolean;

使用案例 ​

ts
import {isFormData} from "jsmethod-extra"

isFormData(new FormData()) // true
isFormData(0) // false
isFormData("") // false
